# Changelog

모든 주요 변경 사항을 기록합니다.

## [1.1.0] - 2025-01-02

### Added
- `/plan` - 스킬 오케스트레이터 (프로젝트 분석 → 스킬 조합 제안)
- `/strategy` - 스킬 활용 전략 가이드
- GitHub 레포 구조화 및 install.sh

### Changed
- `/commands` - 스킬 허브로 개선, 연동 맵 추가
- 모든 스킬 한국어 주석/설명 통일

## [1.0.0] - 2025-01-01

### Added
- `/next-page` - Next.js 14 App Router 페이지 생성
- `/tw-component` - Tailwind + cva 기반 UI 컴포넌트
- `/react-hook` - 커스텀 훅, TanStack Query, Zustand 패턴
- `/ts-form` - react-hook-form + zod 폼 생성
- `/api-route` - Next.js Route Handlers
- `/ftw` - Figma to Web 변환
- `/gen-ai` - AI 이미지/영상 생성
- `/error-doctor` - 에러 분석/해결
- `/commands` - 스킬 목록
